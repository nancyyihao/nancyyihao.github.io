发表于 2018-04-16 分类于 读书 阅读次数： Valine：

1. 为何重构

  • 重构改善软件设计，让代码易于理解
  • 重构帮助找到bug
  • 重构提高编程速度

2. 何时重构

  • 三次法则，事不过三，三则重构
  • 添加新功能时重构
  • 修补错误时重构
  • review代码时重构

3. 坏代码

  • Duplicated Code(重复代码)
  • Long Method(函数过长)
  • Large Class(大类)
  • Long Parameter List(参数过多)
  • So on
